Bubble Plume Data Repository

A unified platform for sourcing raw and processed bubble plume data from key publications.

This repository collects, organizes, and provides access to bubble plume datasets extracted from selected publications. The goal is to make raw and processed data easier to find, download, reproduce, and compare.

Reproducibility notes
