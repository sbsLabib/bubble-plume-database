# Milgram 1983 tabular data and related nondimensional literature-series data

This package contains:

1. `milgram_1983_tabular_selected/`
   Selected dimensional variables extracted from Milgram (1983), Tables 1-4.

2. `li2020_literature_series_nondimensional/`
   Cleaned nondimensional Fannelop & Sjoen (1980) and Milgram & Van Houten (1982) series from the uploaded Excel files used for Li et al. (2020)-style comparison data.

Recommended GitHub locations:

- `csv/milgram_1983_tabular_selected/`
- `csv/li2020_literature_series_nondimensional/`
