# Milgram 1983 selected tabular data

This folder contains selected dimensional tabular data extracted from Milgram (1983), *Mean flow in round bubble plumes*.

## Included variables

- `z_m`: height above gas outlet, m
- `b_smooth_m`: smoothed plume half-width/plume radius at measurement height, m
- `U_smooth_m_s`: smoothed centerline velocity at measurement height, m/s
- `b_midpoint_m`: midpoint average plume half-width/plume radius, m
- `U_midpoint_m_s`: midpoint average centerline velocity, m/s
- `q_m3_s`: gas volume flux, m^3/s
- `gamma`: momentum amplification factor
- `alpha`: entrainment coefficient
- `W`: height-interval weighting term used for evaluating alpha in equation (2.25)

## Table meanings

- Table 1: primary measurement-height data from Bugg Spring, Fannelop & Sjoen, and Milgram & Van Houten.
- Table 2: primary midpoint entrainment data from the same primary datasets.
- Table 3: secondary measurement-height data for Kobus and Topham.
- Table 4: secondary midpoint entrainment data for Kobus and Topham.
