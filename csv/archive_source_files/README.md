# Archived source and development CSV files

This folder contains older split CSV folders and source/development data packages that are retained for provenance and internal review.

The public website should not link directly to these archived folders. Public data downloads should use the paper-level files in:

- `csv/paper_data_downloads/csv/papers/`
- `csv/paper_data_downloads/xlsx/papers/`

For each public paper record, the CSV file is the machine-readable clean data file for reproducible analysis, and the XLSX file is the formatted companion workbook for human inspection.
