# Paper data downloads

This folder provides two download options for each paper.

## CSV files

Machine-readable, official data files:

- `csv/papers/milgram_1983.csv`
- `csv/papers/li_2020.csv`

## XLSX files

Formatted human-readable companion workbooks:

- `xlsx/papers/milgram_1983.xlsx`
- `xlsx/papers/li_2020.xlsx`

The XLSX files include:
- a formatted `Details` sheet with merged headings, color-coded guides, and notes
- a `Data` sheet that preserves the master CSV rows and columns
