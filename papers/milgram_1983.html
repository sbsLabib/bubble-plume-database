Data provenance note

How to interpret this collection

The CSV files on this page represent nondimensionalized comparison data for the Milgram83 literature series. These values were processed from later nondimensional scaling plots, not directly extracted from Milgram’s original tabular data. Milgram (1983) remains the original literature source represented by the series.

Available datasets

Milgram83 cleaned CSV files

Each file contains two columns of nondimensionalized literature-comparison data and can be downloaded directly for inspection or analysis. In this repository, alpha denotes the entrainment coefficient and b_g denotes plume half-width.
