Tabular Data from Milgram 1983

Selected dimensional data extracted from the original tables in “Mean flow in round bubble plumes”

This page describes the selected tabular variables included in the paper-level Milgram 1983 downloads. Public downloads use one official CSV and one formatted XLSX workbook.

Paper synopsis

Milgram (1983) provides tabular plume-property data used to analyze mean-flow behavior in round bubble plumes. The paper-level data file preserves selected dimensional quantities from the original tables, including height above the gas outlet, plume half-width, centerline velocity, gas volume flux, momentum amplification factor, entrainment coefficient, and the weighting term used in the entrainment analysis.

Source publication

Milgram, J. H. (1983). Mean flow in round bubble plumes. Journal of Fluid Mechanics, 133, 345–376.

https://doi.org/10.1017/S0022112083001950

Data downloads

CSV files are provided for reproducible analysis. XLSX files are formatted companion workbooks for easier inspection and include a Details sheet plus a Data sheet.

Data interpretation notes

  • The paper-level CSV combines selected variables from Milgram’s tabular data into one public file.
  • Variables include z_m, b_m, U_m_s, q_m3_s, gamma, alpha, and W where available.
  • The value_basis column indicates whether values are measurement-height smooth values or midpoint averages.
  • Older individual table CSV files are retained only as source/development material and are not public website downloads.
