Tabular Data from Milgram 1983

Selected dimensional data extracted from the original tables in “Mean flow in round bubble plumes”

This page provides download links for selected dimensional tabular data extracted from Milgram (1983). These files are kept separate from the nondimensional comparison datasets.

Paper synopsis

Milgram (1983) provides tabular plume-property data used to analyze mean-flow behavior in round bubble plumes. The selected files on this page preserve key dimensional quantities from the original tables, including height above the gas outlet, smoothed plume half-width, smoothed centerline velocity, gas volume flux, momentum amplification factor, entrainment coefficient, and the weighting term used in the entrainment analysis. These tables complement the nondimensional comparison datasets used elsewhere in the repository.

Primary citation

Source publication

Milgram, J. H. (1983). Mean flow in round bubble plumes. Journal of Fluid Mechanics, 133, 345–376.

https://doi.org/10.1017/S0022112083001950

Interpretation

Data interpretation notes

  • Tables 1 and 3 contain measurement-height values and momentum amplification factor gamma.
  • Tables 2 and 4 contain midpoint values and entrainment coefficient alpha.
  • Tables 1 and 2 are the primary datasets used in Milgram’s analysis.
  • Tables 3 and 4 contain secondary comparison data from Kobus and Topham as analyzed by Milgram.
  • The combined CSV uses b_m and U_m_s as common column names; use the value_basis column to determine whether values are measurement-height smooth values or midpoint averages.
  • These dimensional tabular files should be kept separate from the nondimensional comparison CSVs.
